# 🛒 Cartify — E-Commerce Web App

A full-stack e-commerce application built with the **MERN Stack** featuring AI chat, wishlist, cart, and payment flow.

---

## ✨ Features
- 🔐 JWT Authentication (Login, Signup, Forgot Password)
- 🛍️ Product Browsing, Search & Filter
- 🛒 Cart & Wishlist Management
- 💳 Payment Flow (Card, UPI, Net Banking, COD)
- 🤖 AI Shopping Assistant (Google Gemini)
- ⭐ Product Reviews & Ratings
- 🌙 Dark Mode
- 🔒 Admin Panel

---

## 🧰 Tech Stack
**Frontend:** React.js, Redux Toolkit, Tailwind CSS, React Router  
**Backend:** Node.js, Express.js, MongoDB, Mongoose, JWT  
**AI:** Google Gemini API

---

## ⚙️ Setup

```bash
# Clone repo
git clone https://github.com/Piyush079-jha/Full-Stack-E-Commerce-MERN-APP.git

# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm start
```

### `.env` (backend)
```env
PORT=8080
MONGODB_URI=mongodb://127.0.0.1:27017/cartify
JWT_SECRET=your_secret_key
GEMINI_API_KEY=your_gemini_api_key
```
